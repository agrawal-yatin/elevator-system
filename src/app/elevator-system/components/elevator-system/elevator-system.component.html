Elevator Control System

Welcome to the Elevator Control System POC

This simulation includes the following features:

Supports 10 floors in a multi-story building.
Operates 4 elevators working independently**.
Handles multiple floor requests simultaneously.
Real-time elevator movement and state updates.
Live activity logs for tracking all requests.
Intelligent elevator assignment and direction handling.
Optimized algorithm to pick the closest elevator for each request.

Use the control panel to request elevators and watch them operate in real-time.

** This Elevator system operates on the assumption that all requests are handled through a centralized control panel, with no internal controls inside each of the elevators.
** For POC purpose, I’ve set the default halt time to 1000 ms. It’s configurable in the constants file.
